# 微信公众号运营助手 - 技能使用说明

## 这个技能有什么用？

微信公众号运营助手是一个**本地命令行工具 + AI Agent 技能**，将公众号内容生产全流程自动化：

```
选题 → 写文章 → 配图 → 排版 → 上传素材 → 创建草稿 → 发布
```

你只需要用自然语言告诉 AI 你想写什么，剩下的（模板渲染、AI 配图、HTML 排版、素材上传、草稿创建）全部自动完成。

### 核心价值

| 痛点 | 本技能的解决方案 |
|------|----------------|
| 每次写文章要从零开始 | 8 套专业模板，一键生成文章框架 |
| 找配图耗时耗力 | AI 自动生成封面 + 正文插图 |
| 公众号排版难搞 | Markdown 一键转公众号适配 HTML |
| 素材管理混乱 | 本地素材库 + 自动标签匹配 |
| 发布流程繁琐 | 一条命令创建草稿并发布 |
| 数据分析麻烦 | 命令行直接拉取图文统计数据 |

## 安装与配置

### 1. 克隆项目

```bash
git clone https://github.com/shellaaa1/wechat-official-account-skill.git
cd wechat-official-account-skill
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置环境变量

```bash
cp .env.example .env
```

编辑 `.env` 填入配置：

```env
# 微信公众号（必填）
WECHAT_APP_ID=你的AppID
WECHAT_APP_SECRET=你的AppSecret

# 图片生成 API（可选，用于 AI 配图）
IMAGE_API_BASE_URL=https://api.openai.com/v1
IMAGE_API_KEY=你的API密钥
IMAGE_MODEL=gpt-image-1
```

> ⚠️ 不要把 `.env` 提交到 Git 仓库

### 4. 测试连通性

```bash
# 测试微信 API
python bin/wechat draft list --count 1

# 测试图片生成（需配置 IMAGE_API_KEY）
python bin/wechat image generate --prompt "科技感封面"
```

## 使用方式

### 方式一：AI Agent 对话（推荐）

将技能目录交给 AI Agent（如 CodeBuddy Code），直接用自然语言操作：

**示例对话：**

> **你**：用 tech_article 模板帮我写一篇关于 Python 异步编程的文章，生成封面和插图，然后创建草稿。

> **AI**：我会依次执行：① 渲染模板 → ② 生成插图 → ③ 生成封面 → ④ 创建草稿。请确认文章标题和摘要。

---

### 方式二：命令行直接操作

```bash
# 查看所有模板
python bin/wechat template list

# 渲染模板生成文章
python bin/wechat template render tech_article \
  --var title="Python异步编程入门" \
  --output article.md

# 生成封面图
python bin/wechat image cover --prompt "Python编程，科技感，蓝紫渐变，无文字"

# 为文章自动生成插图
python bin/wechat image illustrate --file article.md --upload

# 创建草稿
python bin/wechat draft create \
  --title "Python异步编程入门" \
  --content-file article_illustrated.md \
  --cover-media-id MEDIA_ID

# 发布
python bin/wechat draft publish DRAFT_MEDIA_ID
```

## 完整工作流示例

### 场景：发布一篇技术文章

```bash
# 第 1 步：用模板生成文章框架
python bin/wechat template render tech_article \
  --var title="从模板到发布：公众号自动化实践" \
  --var summary="介绍如何用命令行工具完成公众号内容生产全流程" \
  --var problem="人工操作耗时长，配图排版繁琐" \
  --var solution="使用 wechat-official-account-skill 自动化工具" \
  --var result="从写作到发布缩短至分钟级" \
  --output articles/my_article.md

# 第 2 步：AI 自动生成正文插图
python bin/wechat image illustrate \
  --file articles/my_article.md \
  --max-images 3 \
  --upload

# 第 3 步：生成封面图并上传微信
python bin/wechat image cover \
  --prompt "公众号封面，自动化工具，科技感，蓝紫渐变，16:9，无文字"

# 第 4 步：创建草稿（使用上一步返回的 media_id）
python bin/wechat draft create \
  --title "从模板到发布：公众号自动化实践" \
  --content-file articles/my_article_illustrated.md \
  --cover-media-id MEDIA_ID \
  --author "AI助手" \
  --digest "用命令行工具完成公众号内容生产全流程"

# 第 5 步：确认内容无误后发布
python bin/wechat draft publish DRAFT_MEDIA_ID
```

## 排版主题

Markdown 转 HTML 时支持 5 种主题：

| 主题 | 风格 |
|------|------|
| `default` | 默认风格 |
| `modern` | 现代科技感 |
| `minimal` | 极简风格 |
| `neon` | 霓虹炫彩 |
| `clean` | 清爽简洁 |
| `warm` | 温暖柔和 |

使用方式：

```bash
python bin/wechat convert \
  --file article.md \
  --theme modern \
  > article.html
```

## 命令速查表

| 需求 | 命令 |
|------|------|
| 查看可用模板 | `python bin/wechat template list` |
| 查看模板详情 | `python bin/wechat template show <name>` |
| 渲染模板生成草稿 | `python bin/wechat template render <name> --var title="..."` |
| Markdown 转公众号 HTML | `python bin/wechat convert --file <md> --theme modern` |
| 上传图片 | `python bin/wechat upload image --file <path>` |
| 生成图片 | `python bin/wechat image generate --prompt "..."` |
| 生成并上传封面 | `python bin/wechat image cover --prompt "..."` |
| 根据文章生成插图 | `python bin/wechat image illustrate --file article.md --upload` |
| 创建草稿 | `python bin/wechat draft create --title ... --content-file ...` |
| 查询草稿 | `python bin/wechat draft list` |
| 发布草稿 | `python bin/wechat draft publish <media_id>` |
| 设置菜单 | `python bin/wechat menu create --file menu.json` |
| 查看粉丝 | `python bin/wechat user list` |
| 获取数据报告 | `python bin/wechat stats article --begin 2026-04-01 --end 2026-04-30` |
